# Portafolio Angular - Julian Esneider Gutierrez Baquero

Este repositorio contiene el proyecto Angular de mi portafolio personal, desarrollado como actividad de la Unidad 1 y 2 de la asignatura de **Desarrollo Web Front-End** en la Fundación Universitaria Compensar.

El proyecto está desarrollado utilizando **Angular**, **TypeScript** y buenas prácticas de **arquitectura MVC**.

## Contenido del Proyecto

El portafolio incluye los siguientes elementos:

- **Página de Inicio:** Sección de bienvenida con información personal, contacto y enlaces a los proyectos.  
- **Página de Contacto:** Formulario de contacto funcional con enlaces de correo y WhatsApp.  
- **Proyectos:**  
  - Unidad 1 
  - Unidad 2 
- **Routing:** Navegación implementada con Angular Router para moverse entre páginas sin recargar el navegador.  
- **Componentes Angular:** Cada sección del portafolio está modularizada mediante componentes reutilizables.
